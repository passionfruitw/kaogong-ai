# 考公AI备考助手 - 产品使用说明书

## 📱 产品简介

考公AI备考助手是一款基于人工智能的公务员考试备考工具，采用苏格拉底式教学法，帮助考生深入理解知识点，而非死记硬背。

**核心功能：**
- 📚 题库练习：5大模块，200+真题
- 🎯 智能刷题：随机抽题，即时反馈
- 🤖 AI辅导：苏格拉底式引导教学
- 💪 强化训练：针对薄弱知识点生成变式题
- 📊 错题管理：自动收集错题，支持重练

## 📖 功能详解

### 1️⃣ 题库管理

**位置：** 首页 → 题库

**功能说明：**
- 查看所有题目（按模块分类）
- 支持模块筛选：常识判断、言语理解、数量关系、判断推理、资料分析
- 点击题目可查看详情、答案和解析

**操作步骤：**
1. 点击顶部导航栏「题库」
2. 使用模块筛选按钮选择科目
3. 点击任意题目查看详情

---

### 2️⃣ 智能刷题

**位置：** 首页 → 刷题

**功能说明：**
- 随机抽取题目进行练习
- 选择答案后即时显示对错
- 答错自动加入错题集
- 支持查看变式题（举一反三）

**操作步骤：**
1. 点击顶部导航栏「刷题」
2. 点击「开始刷题」按钮
3. 阅读题目，选择答案
4. 查看结果反馈
5. 点击「下一题」继续练习

**变式题功能：**
- 答题后点击「举一反三」按钮
- 系统展示同知识点的变式题
- 点击变式题可进入练习模式

---

### 3️⃣ AI辅导（苏格拉底教学）

**位置：** 首页 → AI辅导 → 苏格拉底教学

**功能说明：**
- AI通过提问引导你思考
- 不直接给答案，而是启发理解
- 适合深度学习知识点

**操作步骤：**
1. 点击「AI辅导」→「苏格拉底教学」
2. 选择模块和知识点
3. 点击「开始学习」
4. 在对话框中输入你的想法
5. AI会根据你的回答继续引导

**教学流程：**
- **诊断阶段：** AI了解你的认知水平
- **讲解阶段：** 简明扼要讲解核心要点
- **验证阶段：** 通过追问确认理解
- **总结阶段：** 给出掌握程度评估

---

### 4️⃣ 强化训练

**位置：** 首页 → AI辅导 → 强化训练

**功能说明：**
- 针对薄弱知识点生成专项练习题
- AI自动生成变式题
- 支持批量生成（1/3/5题）

**操作步骤：**
1. 点击「AI辅导」→「强化训练」
2. 查看薄弱知识点列表（按错误率排序）
3. 选择要强化的知识点
4. 选择生成题目数量（1/3/5题）
5. 点击「开始练习」
6. AI生成题目后开始答题

**注意事项：**
- 生成题目需要10-30秒，请耐心等待
- 生成的题目可以加入题库保存

---

### 5️⃣ 错题集

**位置：** 首页 → 错题集

**功能说明：**
- 自动收集所有答错的题目
- 显示你的错误答案和正确答案
- 支持重新练习

**操作步骤：**
1. 点击顶部导航栏「错题集」
2. 查看错题列表
3. 点击任意错题可重新练习
4. 答对后该题会从错题集移除

## 💡 使用技巧

### 高效刷题策略

1. **先做题库练习** → 熟悉题型和知识点分布
2. **使用智能刷题** → 随机练习，检验掌握程度
3. **重点攻克错题** → 定期复习错题集
4. **强化薄弱点** → 使用强化训练针对性提升
5. **AI辅导深化** → 对难点使用苏格拉底教学深入理解

### 变式题的价值

- **举一反三：** 同一知识点的不同考法
- **加深理解：** 避免死记硬背
- **提高应变：** 适应考试中的题目变化

### AI辅导最佳实践

- **主动思考：** 不要直接问答案，先说出自己的想法
- **如实回答：** AI会根据你的水平调整教学
- **多轮对话：** 不懂就继续追问，直到真正理解

## ❓ 常见问题

### Q1: 为什么AI生成题目很慢？
**A:** AI生成题目需要调用大模型API，通常需要10-30秒。生成多道题时会依次生成，请耐心等待。

### Q2: 变式题是什么？
**A:** 变式题是基于原题知识点生成的相似题目，帮助你举一反三，加深理解。

### Q3: 错题集会自动更新吗？
**A:** 是的。每次答错题目会自动加入错题集，答对后会自动移除。

### Q4: 可以导出错题吗？
**A:** 当前版本暂不支持导出，后续版本会添加此功能。

### Q5: AI辅导会直接给答案吗？
**A:** 不会。AI采用苏格拉底式教学，通过提问引导你思考，帮助你自己找到答案。

### Q6: 题库会更新吗？
**A:** 会定期更新真题和热点知识。你也可以通过强化训练生成的题目加入题库。
