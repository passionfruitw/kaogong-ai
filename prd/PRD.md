# 考公AI备考应用 产品需求文档（PRD）

---

## 文档信息

| 项目 | 内容 |
|------|------|
| 产品名称 | 考公AI备考助手 |
| 版本 | v1.0 |
| 产品定位 | 具备AI苏格拉底式教学法的智能考公备考平台 |
| 目标用户 | 首次参加国考/省考的备考考生 |

---

## 一、产品概述

### 1.1 背景

考公备考市场竞争激烈，现有应用（如粉笔、华图在线）功能同质化严重，主要提供题库、视频课程和学习计划等基础功能。随着AI大语言模型技术的发展，用户对智能化、个性化学习辅导的需求日益增长。

### 1.2 产品目标

打造一款具备**AI苏格拉底式教学法**的差异化考公备考应用，通过：
- **AI错题解析**：智能分析错题，提供解题思路
- **举一反三**：基于错题生成同类变式题
- **苏格拉底式教学**：以引导式提问代替直接灌输，真正理解知识点
- **个性化学习追踪**：记录每次学习会话，形成知识掌握画像

### 1.3 产品范围

- **平台**：Web应用（响应式设计，支持PC/平板）
- **考试类型**：国考（中央机关及其直属机构录用公务员考试）、省考（各省公务员录用考试）
- **科目**：行测（言语理解、数量关系、判断推理、资料分析、常识判断）、申论、面试

## 二、用户画像

### 2.1 主要用户群体

#### 用户A：小明，24岁，首次备考
- **背景**：应届大学毕业生，会计专业，零基础备考国考
- **痛点**：
  - 不了解行测各模块考察重点
  - 做题错误率高，答案解析看不懂
  - 缺乏有效的复习计划
- **需求**：
  - AI辅助理解错题背后的知识点
  - 针对性练习薄弱环节
  - 科学的复习节奏提醒

#### 用户B：小红，26岁，在职备考
- **背景**：在职公务员，目标是遴选或省考
- **痛点**：
  - 备考时间碎片化
  - 需要高效利用有限时间
- **需求**：
  - 移动端快速刷题
  - AI总结每日学习重点

### 2.2 用户旅程

```
发现应用 → 注册/登录 → 初始化测评 → 获取学习计划
    ↓
日常学习：每日任务 → 刷题练习 → AI错题解析 → 苏格拉底教学
    ↓
阶段测试 → 进度更新 → 薄弱点强化 → 考前模拟
```

## 三、功能需求

### 3.1 题库与练习模块

#### 功能描述
提供完整的行测、申论、面试题库，支持章节练习、专项训练、模拟考试。

#### 功能列表

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 章节练习 | 按行测五大模块（言语/数量/判断/资料/常识）分章节练习 | P0 |
| 专项训练 | 按细分知识点进行针对性练习 | P0 |
| 模拟考试 | 全真模拟考试，含时间限制、评分 | P1 |
| 历年真题 | 收录近5年国考/省考真题 | P1 |
| 申论批改 | AI辅助批改申论，提供改进建议 | P2 |
| 面试模拟 | AI模拟面试官进行结构化面试练习 | P2 |

#### 题目类型
- 行测：单选题、多选题、判断题
- 申论：归纳概括、对策建议、公文写作、大作文
- 面试：结构化面试问题

### 3.2 错题本与AI解析模块

#### 功能描述
自动收集用户错题，AI智能分析错题原因、涉及的知识点，并提供详细解析。

#### 功能列表

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 自动归集 | 自动将错题收入错题本 | P0 |
| AI解析入口 | 一键调用AI解析当前错题 | P0 |
| 错题分类 | 按模块/知识点/错误类型分类 | P1 |
| 错题重做 | 定期提醒重做错题 | P1 |

#### AI解析流程

```
用户选择错题 → 点击"AI解析" → AI分析步骤：
    1. 识别题目类型和考点
    2. 分析用户可能的错误原因
    3. 给出详细解题步骤
    4. 关联同知识点的其他题目
    5. 引导进入"举一反三"练习
```

#### 解析输出示例

**题目**：下列与"举一反三"意思最相近的是：
A. 触类旁通  B. 事半功倍  C. 举棋不定  D. 三心二意

**AI解析**：
> 本题考查成语理解。"举一反三"意为从一件事物推知其他同类事物。
> - A选项"触类旁通"意为接触到某类事物后，能够理解其他同类事物，与"举一反三"意思相近
> - B选项"事半功倍"形容用力小而收效大
> - C选项"举棋不定"形容犹豫不决
> - D选项"三心二意"形容不专心
> **答案：A**
>
> 这类题目属于言语理解中的**词义辨析**，解题关键是理解成语的字面含义和引申义。

### 3.3 "举一反三"变式题生成模块

#### 功能描述
基于用户的错题，AI自动生成同类变式题，帮助用户彻底掌握知识点。

#### 功能列表

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 自动生成 | 基于错题生成3-5道变式题 | P0 |
| 变式类型 | 同一知识点、不同问法、同类陷阱 | P1 |
| 难度分级 | 基础/进阶/挑战三档 | P1 |
| 按需生成 | 用户可手动触发生成 | P0 |

#### 变式题示例

**原始错题**：下列属于市场经济特征的是：
A. 计划经济  B. 市场在资源配置中起决定性作用  C. 政府完全不管  D. 单一所有制

**AI生成的变式题**：
> **变式1（同一知识点）**："市场经济的基本规律是？"
> - A. 价值规律  B. 供求规律  C. 竞争规律  D. 全部
>
> **变式2（不同问法）**："党的十八届三中全会提出，市场在资源配置中起什么作用？"
> - A. 基础性作用  B. 决定性作用  C. 辅助性作用  D. 引导性作用
>
> **变式3（同类陷阱）**："以下哪项不是市场经济的局限性？"
> - A. 信息不对称  B. 外部不经济  C. 公共物品供给不足  D. 资源配置最优

### 3.4 苏格拉底AI教学模块（核心创新）

#### 功能描述
采用苏格拉底式提问法，通过引导式教学代替直接灌输，让用户真正理解知识点而非死记硬背。

#### 教学理念

苏格拉底教学法的核心是"助产术"（Maieutics），通过不断提问引导用户自己思考、发现答案，从而获得更深刻的学习体验。

#### 对话流程设计

```
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 诊断 —— "先问你知道什么"                            │
├─────────────────────────────────────────────────────────────┤
│  AI：这道题考查的是[知识点名称]，你能先说说你的理解吗？       │
│  用户：回答（前概念/猜测）                                    │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  Step 2: 解释 —— 提供~200字简明讲解                          │
├─────────────────────────────────────────────────────────────┤
│  AI：[~200字精炼讲解]                                        │
│      核心要点：                                              │
│      1. ...                                                 │
│      2. ...                                                 │
│      3. ...                                                 │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  Step 3: 验证 —— 追问确认理解                                │
├─────────────────────────────────────────────────────────────┤
│  AI：类似的问题，如果这样问，你会怎么做？                     │
│  用户：回答                                                  │
│  AI：[判断对错，若错误，继续引导；若正确，进入下一题]        │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  Step 4: 跟踪 —— 记录学习轨迹                                │
├─────────────────────────────────────────────────────────────┤
│  保存本次会话记录                                            │
│  更新知识点掌握状态                                          │
│  调整后续学习策略                                            │
└─────────────────────────────────────────────────────────────┘
```

#### 功能列表

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 诊断式提问 | 询问用户对知识点的初步理解 | P0 |
| 简明讲解 | 200字左右的知识精讲 | P0 |
| 验证式追问 | 通过变式问题验证理解 | P0 |
| 风格适配 | 根据用户反应调整教学深度 | P1 |
| 会话记录 | 完整保存每次教学会话 | P1 |

#### 教学对话示例

**场景**：用户在做错一道关于"货币政策"的题目后，进入苏格拉底教学。

> **AI**：刚才这道题考查的是**货币政策工具**，你能先说说，你对货币政策有哪些了解？哪怕是你的猜测也可以。
>
> **用户**：emm...货币政策好像是国家用来调控经济的，比如利率调整？
>
> **AI**：没错，你说得对！货币政策的核心就是央行通过各种工具来调节货币供应量，从而影响经济。
>
> 不过，货币政策工具主要分为三大类。你知道是哪三类吗？
>
> **用户**：不太清楚...是不是有存款准备金率？
>
> **AI**：对！存款准备金率是其中一种。货币政策工具主要分为：
> 1. **存款准备金率** - 调整商业银行必须存放在央行的资金比例
> 2. **再贴现率** - 央行向商业银行提供贷款的利率
> 3. **公开市场业务** - 央行在市场上买卖有价证券
>
> 这三种工具各有特点：准备金率力度最大但不够灵活；再贴现率是"最后贷款人"工具；公开市场业务则是最常用、最灵活的调控手段。
>
> ---
>
> **AI**：那我来考考你。如果现在经济过热通胀严重，你觉得央行应该采用哪种工具？为什么？
>
> **用户**：应该是...公开市场业务？因为比较灵活，可以随时买卖。
>
> **AI**：很好，你的思路是对的！公开市场业务确实最灵活。但实际上，面对经济过热，央行往往会**同时使用多种工具**：
> - 提高准备金率 → 减少银行可放贷资金
> - 提高再贴现率 → 增加借款成本
> - 卖出有价证券 → 回笼货币
>
> 这些政策组合使用，效果更明显。你现在对货币政策工具的理解是不是更清晰了？
>
> **用户**：是的！原来三类工具是相互配合使用的。
>
> **AI**：很好！来，我们趁热打铁，做一道变式题巩固一下...

## 五、竞品分析

| 竞品 | 优势 | 劣势 |
